## 조항별 핵심 요구사항

### 인력/조직 (제8조~제8조의3)

| 조항 | 핵심 요구사항 |
|------|--------------|
| 제8조 제1항 | 전담 조직 확보, 교육계획 수립, 법규 준수 점검 |
| 제8조 제2항 | 정보기술·정보보호 전문인력 및 예산 확보 |
| 제8조의2 제1항~3항 | 정보보호위원회 설치·운영 |
| 제8조의2 제4항 | 중대 심의·의결사항 이사회 보고 (2025.8.5~) |
| 제8조의3 | CISO 지정, 전략 수립, 사고 예방, 교육 |

### 정보보호 (제15조)

| 항 | 핵심 요구사항 |
|----|--------------|
| 제1항 | 정보보호시스템, 보정작업, 망분리, 파일배포 통합, 전산실 물리분리 |
| 제2항 | 최소 port/기능, 이력 기록, 원격관리 금지 |
| 제3항 | 무선통신망 최소화, 승인, 암호화, AP 점검 |

### 업무연속성 (제23조~제26조)

| 조항 | 핵심 요구사항 |
|------|--------------|
| 제23조 제8항 | 재해복구센터 구축 (2026.2.5 확대: 여신전문금융회사 2조원+, 전자금융업자 2조원+) |
| 제23조 제9항 | 업무지속성 확보, 적정 규모 및 인력 |
| 제23조 제10항 | 연 1회 이상 재해복구전환훈련 |
| 제24조 | 연 1회 비상대응훈련, 결과 금융위원회 보고 |
| 제25조 | 정보처리시스템 성능관리, 사용현황 분석 |
| 제26조 | 직무분리 기준 수립 |

### 보안성심의 (제36조)

| 항 | 핵심 요구사항 |
|----|--------------|
| 제1항 | 신규 전자금융업무, 공동 표준 제정 시 심의 |
| 제2항 | 7일 이내 결과보고서 금융감독원 제출 |
| 제3항 | 금융감독원 개선·보완 요구 가능 |

### 취약점관리 (제37조의2~제37조의4)

| 조항 | 핵심 요구사항 |
|------|--------------|
| 제37조의2 제1항 | 대상: 총자산 2조원+ 또는 상시종업원 300명+ |
| 제37조의2 제2항 | 자체전담반 5인+ (30% 고급기술인력) |
| 제37조의2 제3항 | 소규모: 연 1회, 홈페이지 6개월 1회 |
| 제37조의2 제5항 | 이행계획 수립, 최고경영자 승인·보고 |
| 제37조의3 제1항 | 평가전문기관: 금융보안원, KISA 지정기업, 침해사고대응기관 |
| 제37조의3 제2항 | 결과보고서 금융위원장 제출 |
| 제37조의4 | 침해사고대응기관 업무: 정보공유, 예·경보, 원인분석, 통합보안관제 |
