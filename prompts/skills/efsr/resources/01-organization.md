# 인력/조직 (제8조~제8조의3)

> 근거: 전자금융감독규정 제2025-4호
> 출처: 국가법령정보센터

## 제8조 (인력, 조직, 교육 및 예산)

### 근거 및 시행일

| 항목 | 내용 |
|------|------|
| 조항 | 전자금융감독규정 제8조 |
| 시행일 | **2025.2.5** |
| 상태 | 시행 완료 |

### 핵심 내용

#### 제1항 - 인력 및 조직 운용

금융회사 또는 전자금융업자는 다음 사항을 준수하여야 한다:

| 호 | 요건 | 책임자 |
|----|------|--------|
| 1호 | 정보처리시스템 및 전자금융업무 관련 **전담 조직 확보** | - |
| 2호 | 외부주문등 계약 체결 시 적정성 검토, **내부 조직과 인력 확보** | - |
| 3호 | **매년 교육계획 수립 및 시행** | 정보보호최고책임자 |
| 4호 | 전년도 교육계획 시행 결과 평가, 금년도 교육계획 반영 | 최고경영자 |
| 5호 | **정보보안 관련법규 준수 점검**, 결과 및 보완계획 보고 | 정보보호최고책임자 → 최고경영자 |
| 6호 | 임직원 정보보안 관련법규 위반 시 **제재 기준 및 절차 마련** | 최고경영자 |

#### 제2항 - 전문인력 및 예산

금융회사 또는 전자금융업자는 **정보기술 및 정보보호 분야별 전문성을 갖춘 인력과 충분한 예산을 확보**해야 한다.

### 주요 확인사항

```yaml
체크리스트:
  전담조직:
    - [ ] 정보처리시스템 관련 전담 조직 존재 여부
    - [ ] 전자금융업무 관련 전담 조직 존재 여부
    - [ ] 조직도 및 R&R 문서화 여부

  외부주문:
    - [ ] 외주 계약 체결 전 적정성 검토 절차 수립 여부
    - [ ] 내부 관리 조직 및 인력 확보 여부

  교육:
    - [ ] 당해 연도 교육계획 수립 여부 (CISO 책임)
    - [ ] 전년도 교육 시행 결과 평가 여부 (CEO 책임)
    - [ ] 평가 결과의 금년도 계획 반영 여부

  법규준수:
    - [ ] 정보보안 관련법규 준수 점검 실시 여부
    - [ ] 점검 결과 CEO 보고 여부
    - [ ] 보완계획 수립 및 CEO 보고 여부

  제재:
    - [ ] 임직원 정보보안 법규 위반 시 제재 기준 마련 여부
    - [ ] 제재 절차 문서화 여부

  예산:
    - [ ] 정보기술 분야 전문인력 확보 여부
    - [ ] 정보보호 분야 전문인력 확보 여부
    - [ ] 충분한 예산 확보 및 편성 여부
```

## 제8조의2 (정보보호위원회 운영)

### 근거 및 시행일

| 항목 | 내용 |
|------|------|
| 조항 | 전자금융감독규정 제8조의2 |
| 시행일 | **2025.2.5** (제4항: **2025.8.5**) |
| 상태 | 시행 완료 |

### 핵심 내용

#### 제1항 - 정보보호위원회 설치

금융회사 또는 전자금융업자는 중요 정보보호에 관한 사항을 **심의 및 의결하는 정보보호위원회를 설치 운영**하여야 한다.

#### 제2항 - 구성

| 역할 | 대상 |
|------|------|
| **위원장** | 정보보호최고책임자 (CISO) |
| 위원 | 정보보호업무 관련 부서장 |
| 위원 | 전산운영 및 개발 관련 부서장 |
| 위원 | 준법업무 관련 부서의 장 등 |

#### 제3항 - 심의 및 의결 사항

금융감독원장이 정하는 사항을 심의 및 의결한다.

#### 제4항 - 보고 의무 (2025.8.5 시행)

```
┌─────────────────────────────────────────────────────────────┐
│  정보보호위원회 심의·의결                                      │
│         │                                                    │
│         ▼                                                    │
│  ┌──────────────┐    ┌────────────────────────────────┐      │
│  │ 최고경영자   │    │ 이사회                          │      │
│  │ (CEO)       │    │ (중대한 영향 판단 시)            │      │
│  └──────────────┘    └────────────────────────────────┘      │
│         ▲                       ▲                           │
│         │                       │                           │
│         └───── CISO 보고 ───────┘                           │
└─────────────────────────────────────────────────────────────┘
```

| 보고 대상 | 조건 | 시행일 |
|-----------|------|--------|
| 최고경영자 | 모든 심의·의결사항 | 2025.2.5 |
| **이사회** | 전자금융거래의 안전성 및 신뢰성에 **중대한 영향**을 미친다고 판단하는 사항 | **2025.8.5** |

### 주요 확인사항

```yaml
체크리스트:
  위원회_설치:
    - [ ] 정보보호위원회 설치 여부
    - [ ] 위원회 운영 규정 수립 여부

  구성:
    - [ ] 위원장: CISO 지정 여부
    - [ ] 위원: 정보보호 부서장 포함 여부
    - [ ] 위원: 전산운영/개발 부서장 포함 여부
    - [ ] 위원: 준법 부서장 포함 여부

  심의_의결:
    - [ ] 금융감독원장이 정한 사항에 대한 심의·의결 수행 여부
    - [ ] 심의·의결 기록 보관 여부

  보고_의무:
    - [ ] 심의·의결사항 CEO 보고 여부
    - [ ] 중대 영향 사항 판단 기준 수립 여부
    - [ ] 중대 영향 사항 이사회 보고 여부 (2025.8.5~)
    - [ ] 이사회 보고 기록 보관 여부
```

## 제8조의3 (정보보호최고책임자)

### 근거 및 시행일

| 항목 | 내용 |
|------|------|
| 조항 | 전자금융감독규정 제8조의3 |
| 시행일 | **2025.2.5** |
| 상태 | 시행 완료 |

### 핵심 내용

금융회사 또는 전자금융업자는 **정보보호최고책임자(CISO)**를 지정하여야 하며, CISO의 업무는 다음을 포함한다:

| 호 | CISO 업무 |
|----|----------|
| 1호 | 전자금융거래의 안정성 확보 및 이용자 보호를 위한 **전략 및 계획 수립** |
| 2호 | **정보기술부문의 보호** |
| 3호 | 정보기술부문의 보안에 필요한 **인력관리 및 예산편성** |
| 4호 | 전자금융거래의 **사고 예방 및 조치** |
| 5호 | 전자금융업무 및 그 기반이 되는 정보기술부문 보안을 위한 **자체심의** |
| 6호 | 정보기술부문 보안에 관한 **임직원 교육** |

### CISO 업무 체계

```
┌─────────────────────────────────────────────────────────────┐
│                    CISO 업무 영역                            │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  ┌─────────────┐  ┌─────────────┐  ┌─────────────┐         │
│  │ 전략/계획   │  │ 인력/예산   │  │ 교육        │         │
│  │ (1호)       │  │ (3호)       │  │ (6호)       │         │
│  └─────────────┘  └─────────────┘  └─────────────┘         │
│                                                             │
│  ┌─────────────┐  ┌─────────────┐  ┌─────────────┐         │
│  │ 정보기술    │  │ 사고 예방   │  │ 자체심의    │         │
│  │ 부문 보호   │  │ 및 조치     │  │ (5호)       │         │
│  │ (2호)       │  │ (4호)       │  │             │         │
│  └─────────────┘  └─────────────┘  └─────────────┘         │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 주요 확인사항

```yaml
체크리스트:
  CISO_지정:
    - [ ] CISO 지정 여부
    - [ ] CISO 자격 요건 충족 여부
    - [ ] CISO 직무기술서 수립 여부

  전략_계획:
    - [ ] 전자금융거래 안정성 확보 전략 수립 여부
    - [ ] 이용자 보호 계획 수립 여부

  정보기술_보호:
    - [ ] 정보기술부문 보호 정책 수립 여부
    - [ ] 보호 조치 이행 여부

  인력_예산:
    - [ ] 보안 인력 관리 계획 수립 여부
    - [ ] 보안 예산 편성 여부

  사고_대응:
    - [ ] 사고 예방 계획 수립 여부
    - [ ] 사고 조치 절차 수립 여부
    - [ ] 사고 대응 체계 구축 여부

  자체심의:
    - [ ] 자체심의 절차 수립 여부
    - [ ] 자체심의 수행 기록 여부

  교육:
    - [ ] 임직원 보안 교육 계획 수립 여부
    - [ ] 교육 실시 기록 관리 여부
```
