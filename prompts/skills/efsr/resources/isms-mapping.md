## 2. 금융 특화 요건 (ISMS-P에 없음)

ISMS-P에 동등한 항목이 없거나, 금융권에서 **별도 절차**를 요구하는 항목입니다.

| 구분 | EFSR 조항 | 요건 | 특이사항 |
|------|-----------|------|----------|
| **망분리** | 제15조 제1항 | 내부통신망/외부통신망 분리 | ISMS-P에는 망분리 **의무 없음** |
| **재해복구센터** | 제23조 제8항 | DR센터 구축 의무 | 2026.2.5 확대 (여전사/전금업자 2조원+) |
| **보안성심의** | 제36조 | 신규 전자금융업무 심의 | 금감원 제출 + 개선 요구 가능 |
| **자체전담반** | 제37조의2 제2항 | 취약점평가 5인+ (30% 고급) | ISMS는 인원 요건 없음 |
| **평가전문기관** | 제37조의3 | 금보원, KISA 지정기업 | 금융권 지정 제도 |
| **침해사고대응기관** | 제37조의4 | 정보공유, 통합보안관제 | 금융보안원 중심 체계 |
| **이사회 보고** | 제8조의2 제4항 | 정보보호위원회 → 이사회 | 2025.8.5 시행 |

## 3. 위임 기준

```yaml
# /isms로 위임하는 경우
위임_조건:
  - ISMS-P에 동일/유사 항목 존재
  - 금융 특화 추가 요건 없음
  - 일반적 보안 통제 (암호화, 접근통제, 로그관리 등)

# EFSR에서 직접 작성하는 경우
직접_작성:
  - 금융권 고유 요건 (망분리, DR센터, 보안성심의)
  - ISMS-P보다 강화된 기준 (취약점평가 주기, 자체전담반 인원)
  - 금융권 고유 절차 (금감원 보고, 평가전문기관 지정)
  - 금융권 고유 조직 (정보보호위원회 이사회 보고)
```

## 4. 사용 예시

### 예시 1: 암호화 요건 문의

```
User: /efsr 암호화 요건

Response:
"암호화 일반 요건은 ISMS-P 2.7 참조

금융권 특화 (제15조 제3항):
- 무선통신망 이용 시 암호화 필수
- 금융거래정보 전송 시 E2E 암호화"
```

### 예시 2: 망분리 요건 문의

```
User: /efsr 망분리

Response:
"제15조 제1항 - 망분리 (금융권 특화)

[상세 요건 제공]
- 내부통신망과 외부통신망 분리
- 전산실 물리적 분리
- 파일배포 통합관리

※ ISMS-P에는 망분리 의무 없음"
```

### 예시 3: 취약점평가 문의

```
User: /efsr 취약점 분석평가

Response:
"취약점 점검 일반은 ISMS-P 2.11.2 참조

금융권 강화 요건 (제37조의2):
- 대상: 총자산 2조원+ 또는 상시종업원 300명+
- 자체전담반: 5인+ (30% 고급기술인력)
- 주기: 연 1회 (홈페이지 6개월 1회)
- 평가전문기관: 금융보안원, KISA 지정기업"
```

## 5. ISMS-P 역매핑 (ISMS-P → EFSR)

ISMS-P 인증 준비 시 금융권 추가 요건 확인용.

| ISMS-P 항목 | 관련 EFSR | 금융 추가 요건 |
|-------------|-----------|---------------|
| 1.1.1 경영진의 참여 | 제8조의2 | 정보보호위원회 → 이사회 보고 |
| 1.1.2 최고책임자의 지정 | 제8조의3 | CISO 자격 요건 명시 |
| 1.1.3 조직 구성 | 제8조 | 전담조직 의무 |
| 2.2.2 직무 분리 | 제26조 | - |
| 2.6.1 네트워크 접근 | 제15조 | **망분리 의무** |
| 2.6.6 원격접근 통제 | 제15조 제2항 | **원격관리 금지** 원칙 |
| 2.7.1 암호정책 적용 | 제15조 제3항 | 무선통신 암호화 필수 |
| 2.9.2 성능 및 장애관리 | 제25조 | - |
| 2.10.8 패치관리 | 제15조 제1항 | 보정작업 절차 |
| 2.11.2 취약점 점검 및 조치 | 제37조의2 | 자체전담반, 주기, 전문기관 |
| 2.11.4 사고 대응 훈련 | 제24조 | 금융위 보고 의무 |
| 2.12.1 재해, 재난 대비 | 제23조 | **DR센터 구축 의무** |
| 2.12.2 재해 복구 시험 | 제23조 제10항 | 연 1회 전환훈련 의무 |
