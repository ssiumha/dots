# 전자금융감독규정 개요

## 개요

전자금융감독규정은 금융회사 및 전자금융업자의 정보기술부문 보안 및 전자금융거래의 안전성 확보를 위한 감독 기준입니다.

> **핵심 원칙**: 모든 날짜에는 **법적 근거(조항/고시 번호)**와 **시행일**을 명시합니다.
> 시행일에 따라 적용 기준이 달라지므로, 항상 현행 시행 여부를 확인하세요.

## 법률 체계

전자금융 관련 규제는 다음과 같은 4단계 체계로 구성됩니다.

```
법률 (국회 제정)
  └─ 전자금융거래법
       │
시행령 (대통령령)
  └─ 전자금융거래법 시행령
       │
감독규정 (금융위원회 고시)
  └─ 전자금융감독규정 ← 본 skill 기준
       │
시행세칙 (금융감독원장 고시)
  └─ 전자금융감독규정 시행세칙
```

| 단계 | 명칭 | 제정 기관 | 역할 |
|------|------|----------|------|
| 1단계 | 전자금융거래법 | 국회 | 기본 원칙 및 의무 규정 |
| 2단계 | 전자금융거래법 시행령 | 대통령 | 법률의 위임 사항 구체화 |
| 3단계 | 전자금융감독규정 | 금융위원회 | 감독 기준 및 세부 요건 |
| 4단계 | 전자금융감독규정 시행세칙 | 금융감독원장 | 실무 적용 세부사항 |

## 적용 대상

### 금융회사

| 구분 | 예시 |
|------|------|
| 은행 | 시중은행, 지방은행, 특수은행 |
| 금융투자업자 | 증권사, 자산운용사, 투자자문사 |
| 보험회사 | 생명보험, 손해보험 |
| 신용카드업자 | 신용카드사 |
| 여신전문금융회사 | 캐피탈, 리스사 |

### 전자금융업자

| 구분 | 설명 |
|------|------|
| 전자금융업자 | 전자자금이체업, 직불전자지급수단 발행 및 관리업, 선불전자지급수단 발행 및 관리업, 전자지급결제대행업 등 |
| 전자금융보조업자 | 금융회사 또는 전자금융업자를 위해 전자금융업무를 보조하는 자 |

## 2025년 개정사항

> 근거: 금융위원회고시 제2025-4호
> 출처: 국가법령정보센터, 김앤장 법률사무소

### 시행일 체계

| 시행일 | 조항 | 내용 |
|--------|------|------|
| **2025.2.5** | 대부분 조항 | 일반 시행 |
| **2025.8.5** | 제8조의2 제4항 | 정보보호위원회 심의 및 의결사항 이사회 보고 |
| **2026.2.5** | 제5조 제1항, 제2항 | 책임이행보험 한도 상향 |
| **2026.2.5** | 제23조 제8항 | 재해복구센터 설치 의무 확대 |

| 구분 | 기준 | 비고 |

| 은행 | 전체 | 기존 |
| 금융투자업자 | 전체 | 기존 |
| 신용카드업자 | 전체 | 기존 |
| 보험회사 | 전체 | 기존 |

## ISMS-P와의 관계

전자금융감독규정과 ISMS-P 인증은 상호 보완 관계입니다.

### 비교

| 구분 | 전자금융감독규정 | ISMS-P |
|------|-----------------|--------|
| 성격 | 법적 의무 (규제) | 인증 제도 (자율/의무) |
| 근거 | 전자금융거래법, 금융위원회고시 | 정보통신망법, 개인정보보호법 |
| 감독기관 | 금융위원회, 금융감독원 | 과기정통부, 개인정보보호위원회 |
| 적용대상 | 금융회사, 전자금융업자 | 정보통신서비스 제공자, 개인정보처리자 |
| 초점 | 전자금융거래 안전성 | 정보보호 및 개인정보보호 관리체계 |

### 중복 적용

금융회사는 **전자금융감독규정**과 **ISMS-P** 양쪽 모두 적용받을 수 있습니다.

| 의무 | 전자금융감독규정 | ISMS-P |
|------|-----------------|--------|
| 은행 | 필수 | 매출/이용자 기준 충족 시 필수 |
| 대형 핀테크 | 필수 | 의무대상 해당 시 필수 |
| 일반 전자금융업자 | 필수 | 자율 인증 가능 |

### 매핑 관계

전자금융감독규정의 주요 조항은 ISMS-P 인증기준과 다음과 같이 매핑됩니다.

| 전자금융감독규정 | ISMS-P 인증기준 |
|-----------------|----------------|
| 제8조 (인력, 조직, 교육) | 1.1.1 경영진 참여, 1.2.4 교육 및 훈련 |
| 제8조의2 (정보보호위원회) | 1.1.1 경영진 참여, 1.1.3 조직 구성 |
| 제8조의3 (CISO) | 1.1.2 최고책임자 지정 |
| 제15조 (해킹 방지) | 2.6.1 네트워크 보안, 2.10.1 보안시스템 운영 |
| 제23조 (재해복구) | 2.12.1 재해/재난 대비, 2.12.2 재해복구 시험 |
| 제37조의2 (취약점 분석평가) | 1.4.1 법적 요구사항 검토, 2.11.2 취약점 점검 |
