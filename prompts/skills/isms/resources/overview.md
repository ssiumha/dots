# ISMS-P 인증 가이드

## 개요
정보보호 및 개인정보보호 관리체계(ISMS-P) 인증 준비를 위한 참고 자료입니다.

> **핵심 원칙**: 법률/규정 관련 내용은 반드시 **근거(고시 번호)**와 **시행일**을 명시합니다.
> 시간에 따라 기준이 변경될 수 있으므로, 항상 현행 기준 여부를 확인하세요.

## 법적 근거
- 정보통신망법 제47조, 제47조의2, 제47조의7
- 개인정보보호법 제32조의2
- 정보보호 및 개인정보보호 관리체계 인증 등에 관한 고시

## 용어 설명

| 용어 | 설명 |
|------|------|
| **고시** | 행정기관(과기부, 개보위)이 발령하는 공식 규정. 법률의 세부사항을 정함 |
| **[별표 7]** | 고시 문서의 첨부 표 7번. ISMS-P 인증기준 101개 항목이 수록됨 |
| **개정일** | 고시가 수정·공포된 날짜 (본 skill 기준: 2023.10.5) |
| **시행일** | 법령이 효력을 발생하는 날짜. 개정일과 동일하거나 유예기간 후 적용 |

※ 본 skill의 인증기준은 **2023.10.5 개정 고시 [별표 7]** 기준입니다.

## 인증 대상
### 의무 인증 대상
| 구분 | 기준 |
|------|------|
| ISP | 서울 및 모든 광역시에서 서비스 제공 |
| IDC | 집적정보통신시설 사업자 |
| 매출 기준 | 정보통신서비스 매출 100억원 이상 |
| 이용자 기준 | 일평균 이용자 100만명 이상 |
| 특정 규모 | 매출/세입 1,500억원 이상 + 상급종합병원/대학(1만명+) |

### 자율 인증 대상
모든 개인정보처리자 및 정보통신서비스 제공자

## ISMS vs ISMS-P 비교

| 구분 | ISMS | ISMS-P |
|------|------|--------|
| 정식명칭 | 정보보호 관리체계 | 정보보호 및 개인정보보호 관리체계 |
| 적용 영역 | 영역 1 + 영역 2 | 영역 1 + 영역 2 + 영역 3 |
| 인증기준 항목 수* | 80개 | 101개 |
| 개인정보보호 | 미포함 | 포함 (21개 항목) |
| 적합 대상 | 개인정보 미보유 기관 | 개인정보 처리 기관 |

*인증기준 항목 = 인증심사의 기본 평가 단위
세부점검항목 포함 시 232개

## 인증 유형
| 유형 | 영역 | 항목 수 |
|------|------|--------|
| ISMS | 관리체계 + 보호대책 | 80개 |
| ISMS-P | 전체 3개 영역 | 101개 |

## 인증기준 구성 (ISMS-P 공식 기준)

| 영역 | 분야 | 인증기준 항목 |
|------|------|------|
| 1. 관리체계 수립 및 운영 | 4개 | 8개 |
| 2. 보호대책 요구사항 | 12개 | 64개 |
| 3. 개인정보 처리단계별 요구사항 | 5개 | 21개 |
| **합계** | **21개** | **101개** |

※ 세부점검항목 포함 시 232개

**NOTE**: 본 skill의 리소스 파일은 ISMS-P 정식 인증 기준(101개 항목)으로 구성되어 있습니다.
ISMS 인증(80개 항목, 개인정보 제외)을 준비하는 경우 영역 1, 2만 참조하세요.

## 인증 절차
1. ISMS-P 구축 (2개월+ 운영 이력)
2. 인증 신청 (심사 8주 전)
3. 심사 준비 (심사 6주 전)
4. 인증심사 (1~2주)
5. 보완조치 (100일 이내)
6. 인증위원회 심의
7. 인증서 발급

## 유효기간
- 최초/갱신: 3년
- 사후심사: 매년 1회 이상

## 인증기관
- **법정**: 한국인터넷진흥원(KISA)
- **지정**: 금융보안원(FSI)

## 개정 이력

### 2025.12 강화 방안 (법 개정 추진 중)

> 상태: 추진 중 (미시행)
> 출처: 과기정통부·개보위 관계부처 대책회의 (2025.12.6)

**예정 변경 사항**:
- 통신사, 대형 플랫폼 ISMS-P **의무화**
- 강화된 인증기준 적용 (국민 파급력 큰 기업 대상)
- 심사 강화: 예비심사 핵심항목 선검증, 기술심사·현장실증 강화
- 사후관리 강화: 유출사고 시 특별 사후심사, 중대결함 시 인증 취소
- AI 등 신기술 심사역량 교육

※ 법제화 시 고시 [별표 7] 개정 예상. 확정 시 본 skill 업데이트 필요.

### 2024년 변경 사항

**2024.7 - 인증제도 안내서 개정**
> 고시 [별표 7] 변경 없음 (안내서만 개정)

- 법령 개정 반영
- 신규 심사기관 추가, 인증 유형 변경
- 오탈자 수정

**2024.3.15 - 개인정보보호법 2차 개정 시행**
> 고시 [별표 7] 변경 없음 (법률만 개정)

- 자동화된 결정에 대한 정보주체 권리 보장
- 개인정보보호 책임자 자격요건 강화
- 영역 3 항목 해석 시 참고 필요

### 2023.10 개정 (현행 기준)

> 근거: 과기부고시 제2023-33호, 개보위고시 제2023-8호

**개정 경위**:
```
2023.9.15  개인정보보호법 개정 시행 (법률)
    ↓      법률 변경에 맞춰 고시도 개정 필요
2023.10.5  고시 [별표 7] 개정·시행 ← 본 skill 기준
    ↓
2023.10.31 KISA 세부점검항목 공지 (실무 가이드)
2023.11.8  인증기준 안내서 발행
```

**변경 사항**: 신설 9항목, 수정 17항목

**신설 항목**:
- 2.3.4 업무환경 보안
- 2.4.4 비밀번호 관리
- 2.5.4 인터넷 접속 통제
- 3.1.1 개인정보 수집·이용
- 3.1.4 민감정보 및 고유식별정보 처리 제한
- 3.1.6 영상정보처리기기 설치·운영
- 3.2.4 개인정보 목적 외 이용 및 제공
- 3.2.5 가명정보 처리
- 3.3.1 개인정보 제3자 제공

### 2018.11 통합 (ISMS + PIMS → ISMS-P)

> 근거: 과학기술정보통신부고시 제2018-82호, 방송통신위원회고시 제2018-16호
> 시행: 2018년 11월 7일

- 기존 ISMS(정보보호)와 PIMS(개인정보보호) 인증을 단일 체계로 통합
- **통합 후 번호 체계 일원화**: ISMS/ISMS-P 모두 동일한 [별표 7] 사용
  - ISMS: 영역 1, 2 적용 (80개)
  - ISMS-P: 영역 1, 2, 3 적용 (당시 102개 → 현행 101개)
