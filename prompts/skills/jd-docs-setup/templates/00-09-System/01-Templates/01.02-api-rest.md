## 엔드포인트

### `GET /resource`

리소스 목록 조회

**요청**

| 파라미터 | 타입 | 필수 | 설명 |
|----------|------|------|------|
| page | int | N | 페이지 번호 (기본: 1) |
| limit | int | N | 페이지 크기 (기본: 20) |

**응답**

```json
{
  "data": [],
  "meta": {
    "total": 100,
    "page": 1,
    "limit": 20
  }
}
```

**에러 코드**

| 코드 | 설명 |
|------|------|
| 400 | 잘못된 요청 |
| 401 | 인증 필요 |
| 404 | 리소스 없음 |

---

### `GET /resource/{id}`

단일 리소스 조회

**경로 파라미터**

| 파라미터 | 타입 | 설명 |
|----------|------|------|
| id | string | 리소스 ID |

**응답**

```json
{
  "data": {
    "id": "123",
    "name": "example"
  }
}
```

---

### `POST /resource`

리소스 생성

**요청 본문**

```json
{
  "name": "string",
  "description": "string"
}
```

**응답** (201 Created)

```json
{
  "data": {
    "id": "123",
    "name": "example"
  }
}
```
