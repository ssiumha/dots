# Cross-Domain Reference

도메인 간 교차 참조가 필요한 경우 이 파일을 사용합니다.

## EFSR ↔ ISMS-P 매핑

**상세 매핑**: `efsr/isms-mapping.md`

| EFSR 조항 | ISMS-P 항목 | 관계 |
|-----------|-------------|------|
| 제8조 (인력/조직) | 1.1.x (관리체계 수립) | 참조 (ISMS 기반 + 금융 특화) |
| 제15조 (정보보호) | 2.6.1, 2.10.1 | 참조 + 금융 특화 (망분리 의무) |
| 제23조~26조 (업무연속성) | 2.12.1 (재해복구) | 참조 + 금융 특화 (DR센터 의무) |
| 제36조 (보안성심의) | - | 금융 특화 (ISMS-P에 없음) |
| 제37조의2~4 (취약점) | 2.11.2 (취약점 점검) | 참조 + 금융 특화 (자체전담반) |

**금융 특화 요건** (ISMS-P에 없는 것):
- 망분리 의무 (제15조)
- DR센터 구축 (제23조)
- 보안성심의 + 금감원 제출 (제36조)
- 자체전담반 5인+, 30% 고급인력 (제37조의2)
- 이사회 보고 (제8조의2)
