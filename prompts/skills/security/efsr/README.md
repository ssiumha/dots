# 전자금융감독규정 준수 지원

금융회사 및 전자금융업자의 전자금융감독규정(금융위원회고시 제2025-4호) 준수를 위한 조항 조회, 체크리스트 생성, 이행 계획 수립을 지원한다.

## Quick Reference

```
/security efsr 제15조                # 조항 번호로 조회
/security efsr 망분리               # 키워드로 관련 조항 검색
/security efsr 제15조 체크리스트     # 체크리스트 생성
/security efsr 정보보호 계획         # 이행 계획 수립
```

## Workflows

### 1. 조항 조회

조항 번호(제15조) 또는 키워드(망분리, CISO)로 관련 규정을 검색한다.
- 조항 번호 -> 파일 매핑표로 직접 탐색
- 키워드 -> Grep으로 전체 검색
- 결과: 핵심 요구사항, 주요 확인사항(상위 3개), 시행일, ISMS-P 참조, 파일 경로

### 2. 체크리스트 생성

특정 조항의 확인사항을 실행 가능한 체크리스트로 변환한다.
- 필수("~해야 한다") / 권장(권고 사항) / N/A(해당 없음) 분류
- 출력: `./templates/checklist.md` 형식

### 3. 이행 계획 수립

분야별 단계적 이행 계획을 생성한다.
- 의존성 분석: 제8조 조직 구성이 모든 분야의 선행 항목
- Phase: 현황 분석 -> 기술적 조치 -> 교육/훈련 -> 검증/모의점검
- 출력: `./templates/plan.md` 형식

### 4. ISMS-P 연계 조회

EFSR 조항과 ISMS-P 항목 간 매핑을 제공한다.
- 참조: ISMS-P에 동일/유사 항목 존재 -> `../isms/` 참조 안내
- 직접: 금융 특화 요건(ISMS-P에 없음) -> EFSR에서 상세 제공
- 매핑 데이터: `./isms-mapping.md`

## 금융 특화 요건 (ISMS-P에 없음)

| 구분 | EFSR 조항 | 요건 |
|------|-----------|------|
| 망분리 | 제15조 제1항 | 내부/외부통신망 분리 의무 |
| DR센터 | 제23조 제8항 | 재해복구센터 구축 의무 |
| 보안성심의 | 제36조 | 신규 전자금융업무 심의 + 금감원 제출 |
| 자체전담반 | 제37조의2 제2항 | 취약점평가 5인+ (30% 고급기술인력) |
| 이사회보고 | 제8조의2 제4항 | 정보보호위원회 -> 이사회 보고 |

## 중요 원칙

1. **공식 문서만 참조**: 외부 검색 없이 skill 내부 리소스만 사용
2. **원본 위치 명시**: 모든 출력에 파일 경로 포함
3. **시행일 명시**: 조항별 시행일(시행 완료/예정) 반드시 표시
4. **ISMS 위임**: 금융 특화 요건만 직접, 일반 보안은 `../isms/` 참조 안내
