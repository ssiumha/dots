# ISMS-P 인증 준비 지원

ISMS-P (정보보호 및 개인정보보호 관리체계) 인증 준비를 위한 항목 조회, 체크리스트 생성, 이행 계획 수립을 지원합니다.

## Quick Reference

```
/security isms 접근통제              # 키워드로 관련 항목 검색
/security isms 2.4                  # 항목 번호로 검색
/security isms 2.4 체크리스트         # 체크리스트 생성
/security isms 인적보안 계획          # 이행 계획 수립
```

## Workflows

### 1. 항목 조회

키워드 또는 항목 번호로 관련 인증기준을 검색한다.

- **항목 번호** (예: 2.4, 2.4.1): 해당 파일 직접 탐색
- **키워드** (예: 접근통제): Grep으로 전체 검색
- "체크리스트" 포함 시 Workflow 2, "계획" 포함 시 Workflow 3으로 전환

검색 결과: 항목 번호, 항목명, 인증기준 요약, 주요 확인사항 (상위 3개), 파일 경로

### 2. 체크리스트 생성

특정 분야의 실행 가능한 체크리스트를 생성한다.

- 대상: 항목 번호(2.4), 분야명(인적보안), 단일 항목(2.4.1)
- "주요 확인사항"에서 체크박스 항목 추출
- 분류: **필수**(인증기준 요구), **권장**(자체 추가), **N/A**(해당 없음)
- 템플릿: `./templates/checklist.md`

### 3. 이행 계획 수립

특정 분야의 단계별 이행 계획을 생성한다.

- 의존성 분석: 공통 선행 항목(1.1.5 정책, 1.2.1 자산) 확인
- 난이도: 쉬움(정책만), 보통(기존 도구 활용), 어려움(신규 시스템)
- Phase: 정책/절차 수립 → 기술적 구현 → 교육/훈련 → 검증/테스트
- 템플릿: `./templates/plan.md`
- 전체 로드맵: `./preparation-guide.md`
